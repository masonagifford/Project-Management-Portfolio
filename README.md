## Portfolio Projects

### 1. Project Office Green
Developed a comprehensive project plan for launching Plant Pals, an office plant delivery service designed for high-volume clients. The project included key project management deliverables such as a project charter, work breakdown structure (WBS), Gantt chart, and task and milestone network diagram to support structured planning and execution.

**Key Skills:** 
- Planning
- Scheduling and Timeline Development
- Risk Management
- Work Breakdown Structure (WBS) Development

### 2. Community Center
Developed a project proposal to repurpose a vacated middle school into a multi-use community center offering after-school programs, adult education, a library space, and a community garden. The project included planning for initial staffing structure (1 director and 2 program coordinators) with a roadmap for future staffing expansion. All planning activities were developed within a constrained $250,000 budget, incorporating scope definition, resource planning, and program prioritization.

**Key Skills:** 
- Project Planning
- Budgeting
- Scope Definition
- Resource Planning
- Stakeholder Alignment
